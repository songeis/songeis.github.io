# 1.6 链表的遍历

我们昨天说到了链表的结构，在 JavaScript 中的实现是一个多层的对象，然后我们又说过我们需要一个计数器知道到底有多少个元素。

今天就开始尝试着实现吧。

# count 的实现

默认情况下

count=0

每次往里添加一个元素，count 就+1

根据昨天的代码我们在 constructor 的里设置

constructor(){
	this.count = 4;
}

我们可以将遍历代码写成如下的形式：

getElementAt() {
    // 指针当在的位置
    let current
    for (let i = 0; i < this.count; i++) {
      // 指针默认在第一个元素上
      current = this.head
      // 指针往下走，指向就是 current 的 next 那当前就是那个 current
      this.head = current.next
      console.log(current)
    }
  }

但回顾一下上次的代码，我们在添加节点的时候方式很蠢。

  addNode() {
    this.firstClassCarriages.next = this.secondClassCarriages
    this.secondClassCarriages.next = this.thirdClassCarriages
    this.thirdClassCarriages.next = this.fourthClassCarriages
    this.head = this.firstClassCarriages
    // this.head = this.secondClassCarriages
    return this.head
  }

每次的结点挂载我们都是手动赋值。这一点我们要怎么改善一下呢。

class Train {
  constructor() {
    this.head = {
      value: undefined,
      next: null
    }
    // counter
    this.count = 0
  }
  addNode(element) {
    // 挂载第一个元素：没有节点的时候count=0，这个时候 head 可以直接指向新添加的元素
    if (this.count === 0) {
      this.head = element
    } else {
      let current
      current = this.head
      while (current.next != null) {
        // 指针开始移动
        current = current.next
      }
      current.next = element
      // console.log(current)
    }
    this.count++;
  }
  getElementAt() {
    // 指针当在的位置
    let current
    for (let i = 0; i < this.count; i++) {
      // 指针默认在第一个元素上
      current = this.head
      // 指针往下走，指向就是 current 的 next 那当前就是那个 current
      this.head = current.next
      console.log(current)
    }
  }
}
let firstClassCarriages = {
  value: "a",
  next: null
}
let secondClassCarriages = {
  value: "b",
  next: null
}
let thirdClassCarriages = {
  value: "c",
  next: null
}
let fourthClassCarriages = {
  value: "d",
  next: null
}
let hexie = new Train()
// console.log(hexie)
hexie.addNode(firstClassCarriages)
// 挂第二个车厢
hexie.addNode(secondClassCarriages)
// 挂第三个车厢
hexie.addNode(thirdClassCarriages)
// 挂第四个车厢
hexie.addNode(fourthClassCarriages)
hexie.getElementAt()
